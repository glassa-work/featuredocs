# Per-message model attribution

When a single conversation hops between models — say you started on a fast cheap one and switched to a stronger one for a hard question — it can be hard to remember which reply came from where.

Each model reply now shows a small label with the model that produced it, right next to the message. At a glance you can tell which answers came from which brain, useful for comparing quality or remembering which provider handled a tricky case.
