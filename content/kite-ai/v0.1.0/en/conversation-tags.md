# Conversation tags with multi-select filter

Tags are a flexible way to organise conversations beyond projects. A chat can have any number of tags — `work`, `recipes`, `travel`, `read-later` — and you can filter the drawer by one or more of them.

## Adding tags

Open a conversation's menu and pick "Tags". Add new tags by typing them, or pick from existing ones. Tags appear as small chips next to the conversation title in the drawer.

## Filtering

Tap the filter button at the top of the drawer and select one or more tags. The list narrows to conversations that match all selected tags. Clear the filter to see everything again.
