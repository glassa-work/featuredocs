# Copy conversation as markdown

Open a conversation's menu and pick "Copy as markdown" to put the whole thread on your clipboard, formatted as clean markdown with each turn clearly labelled.

Paste it into a notes app, a doc, an email, or a GitHub issue. Code blocks, lists, and other formatting come through intact, so the result reads well without cleanup. It is the simplest way to share a chat with someone else, or to archive a useful exchange outside the app.
