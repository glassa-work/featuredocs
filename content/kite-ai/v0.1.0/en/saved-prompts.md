# Saved prompts with slash autocomplete

If you find yourself typing the same prompt over and over — "Translate this to French", "Summarize as bullet points", "Explain like I'm five" — save it once and recall it with a slash command.

## Saving a prompt

Open Settings, go to Saved prompts, and add one. Give it a short slug (like `tldr`) and the full prompt text it should expand to.

## Using a prompt

In the composer, type `/` to open the autocomplete menu and start typing the slug. Pick one and the full prompt is dropped into the composer, ready to send or edit further.
