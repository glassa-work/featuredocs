# System prompt presets library

Setting a good system prompt is half the battle. Kite ships with a small library of ready-made presets — things like "Concise expert", "Friendly tutor", "Code reviewer", "Brainstorming partner" — that you can apply to a project or a single conversation in one tap.

## Using a preset

When editing a system prompt, open the presets sheet and pick one. The text drops into the editor; tweak it before saving if you like.

## Saving your own

Like a custom prompt you wrote? Save it to the library so it is one tap away next time. Your custom presets show up alongside the built-in ones.
