# Projects

Projects let you group related conversations under a shared workspace. Every chat inside a project starts with the same custom system prompt, so you do not have to re-explain context each time.

## Creating a project

Open the project picker at the top of the drawer and tap "New project". Give it a name and a system prompt — for example a project called "Recipes" with the instruction "You are a friendly chef who keeps answers short and uses metric units."

## Switching context

Switch projects from the same picker. Conversations from other projects are hidden from the drawer until you switch back, keeping each context clean.
