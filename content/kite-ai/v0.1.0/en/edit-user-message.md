# Edit and resend a user message

Made a typo, missed a detail, or just want to nudge the model in a different direction? Tap any message you sent and pick "Edit". The text drops back into the composer for you to change.

When you send the edited version, Kite removes the original message and the reply that followed it, then asks the model again with your new prompt. Earlier turns stay intact, so the rest of the conversation context is preserved.
