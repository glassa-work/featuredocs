# Searchable model picker

Once you have several providers configured, the model list can grow long. The picker now has a search box at the top — start typing and the list filters in real time, matching against both model names and provider names.

So `haiku` jumps straight to Anthropic's Haiku models, `groq` filters down to everything you can reach through Groq, and so on. Combined with model favorites, it makes picking the right tool for the job nearly effortless.
