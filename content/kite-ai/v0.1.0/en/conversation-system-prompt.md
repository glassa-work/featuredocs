# Per-conversation custom system prompt

Projects let you share a system prompt across many chats, but sometimes you just want one conversation to behave differently. Open a chat's menu and pick "System prompt" to set or edit one for that conversation alone.

The custom prompt overrides the project-level prompt for this chat only. Clear it to fall back to the project default. This is the lightest way to give a single conversation a specific role — a reviewer, an interviewer, a translator — without setting up a whole new project.
