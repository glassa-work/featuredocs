# Fork chat from a specific message

Want to explore an alternate path through a conversation without losing the original? Tap any earlier message and pick "Fork from here". Kite creates a new conversation that contains every message up to and including the one you picked.

The new chat lands at the top of the drawer; the original is left untouched. From there you can ask a different follow-up, switch models, or change the system prompt and see where things go. It is the cleanest way to branch a discussion when you are not sure which direction is best.
