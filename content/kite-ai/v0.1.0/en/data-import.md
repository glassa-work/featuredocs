# Import data from JSON backup

Got a Kite export file from another device, or restoring after a reinstall? Open Settings, pick "Import data", and choose the JSON file. Kite reads it and brings back every conversation, project, saved prompt, tag, and setting it contains.

Existing data on the device is merged rather than wiped, so you can safely import on a phone that already has chats on it. Provider API keys are not included in exports for security — you will need to re-enter those.
