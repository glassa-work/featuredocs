# Regenerate with a different model

Not happy with a reply? Open the message menu and pick "Regenerate with…" to choose a different model and try again. The previous reply is replaced by the new one.

This is great for comparing models on the same prompt, falling back to a stronger model when a cheaper one struggles, or trying a different provider when one is having a bad day. Combined with per-message model attribution, you can see exactly which model produced the answer you ended up with.
