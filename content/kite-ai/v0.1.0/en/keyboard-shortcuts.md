# Keyboard shortcuts

If you use Kite AI with a hardware keyboard — on iPad, on a Mac, or on a phone with one paired — three shortcuts will save you a lot of taps:

- **Cmd-Return** — send the current message.
- **Cmd-N** — start a new conversation.
- **Cmd-K** — focus the conversation search box in the drawer.

Plain Return still inserts a newline in the composer, so multi-line prompts behave the way you expect.
