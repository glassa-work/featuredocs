# Markdown rendering

Kite AI renders model replies as proper formatted markdown rather than a wall of plain text. Headings, bold and italic text, bullet and numbered lists, blockquotes, tables, links, and inline and fenced code blocks all show up the way the model intended.

This makes long answers — tutorials, recipes, code walk-throughs, comparison tables — much easier to scan, and means you can copy text out with the formatting preserved.
