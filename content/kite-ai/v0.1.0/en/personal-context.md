# Personal context (memory)

Tired of telling every new model your name, where you live, or what you do for work? Personal context is a small block of facts about you that Kite quietly attaches to every conversation, on every provider.

## Setting it up

Open Settings and find "Personal context". Write a short bio in plain English — anything you would normally have to repeat. For example: "My name is Sam, I live in Berlin, I'm vegetarian, and I prefer answers in metric units."

## How it's used

The text is added to the system prompt of every conversation, so any model you talk to already knows the basics. Edit or clear it any time.
