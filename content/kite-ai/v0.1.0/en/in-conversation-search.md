# In-conversation message search

Long chats accumulate. To find a specific message inside an open conversation, open the chat menu and pick "Search". A search bar appears at the top.

Type a few words; matches are highlighted and the chat scrolls to each one. Use the up and down arrows to step through results. Close the search bar to return to the normal view. Pair this with the drawer search to first find the right conversation, then the right message inside it.
